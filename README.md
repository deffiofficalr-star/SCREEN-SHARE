# 🖥️ Screen Share

**Трансляция экрана в реальном времени через браузер.**

---

## 📌 Что это?

Screen Share — это программа для захвата и трансляции экрана компьютера в реальном времени. Достаточно запустить сервер и открыть ссылку в браузере — можно смотреть экран с любого устройства без установки дополнительного ПО.

---

## ✨ Возможности

- 🖥️ Захват всего экрана или отдельного монитора
- 📱 Просмотр в любом браузере (ПК, телефон, планшет)
- ⚡ Настройка качества (FPS, разрешение, сжатие)
- 👥 Поддержка нескольких зрителей одновременно
- 🔄 Обновление в реальном времени
- ⛶ Режим полного экрана

---

## 🚀 Быстрый старт

```bash
git clone https://github.com/deffiofficalr-star/SCREEN-SHARE.git
cd SCREEN-SHARE
pip install -r requirements.txt
python app.py
```

Открой в браузере: http://localhost:5000

---

📱 Как использовать

1. Запусти сервер на компьютере, экран которого хочешь транслировать
2. Открой http://localhost:5000 — страница управления
3. Настрой качество и нажми "Начать трансляцию"
4. Открой http://localhost:5000/stream на любом устройстве (телефон, планшет, другой ПК)
5. Смотри экран в реальном времени

---

🔧 Настройки

Параметр Описание
Монитор Выбор монитора для захвата
Качество Сжатие JPEG (10-100%)
FPS Кадров в секунду (1-60)
Разрешение Размер изображения

---

🌐 Доступ через интернет

Чтобы транслировать экран через интернет, используй ngrok:

```bash
ngrok http 5000
```

Получишь публичную ссылку для просмотра с любого устройства.

---

🛠 Технологии

· Python 3.8+
· Flask + Socket.IO
· OpenCV / mss (захват экрана)
· HTML/CSS/JS (веб-интерфейс)

---

⚠️ Важно

· Программа работает в локальной сети
· Для интернет-доступа используй ngrok или проброс портов
· Чем выше качество — тем больше нагрузка на процессор
